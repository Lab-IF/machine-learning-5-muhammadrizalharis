# Pertemuan 01  
Dasar Algoritma untuk Machine Learning

## Materi
Pada pertemuan ini dipelajari dasar penggunaan library Python untuk machine learning, yaitu:
- NumPy untuk pengolahan data numerik dan operasi matriks.
- Pandas untuk membuat dan mengelola data dalam bentuk DataFrame.
- Matplotlib untuk membuat visualisasi data.
- Pengenalan dataset Wine dari scikit-learn.

## Tugas
1. Membuat 20 angka acak menggunakan NumPy dan menghitung statistik dasar (mean, median, std, min, max).
2. Mengubah array menjadi bentuk matriks dan menampilkan dimensinya.
3. Membuat DataFrame data mahasiswa (nama, nilai, status lulus/tidak lulus).
4. Menghitung rata-rata kelas, jumlah mahasiswa lulus, dan mencari tiga nilai tertinggi.
5. Membuat visualisasi bar chart dan histogram.
6. Memuat dan mengeksplorasi dataset Wine (shape, head, describe).

Notebook lengkap terdapat pada file `.ipynb` di folder ini.
